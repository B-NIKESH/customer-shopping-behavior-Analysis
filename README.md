# 🛍️ Customer Shopping Behaviour Analysis

## 📌 Project Overview

This project analyzes customer shopping behaviour using Excel and Power BI. The objective was to identify purchasing patterns, understand customer preferences, and generate actionable insights to support business decision-making.

The project was completed as part of my Data Analytics learning journey to strengthen practical skills in data cleaning, analysis, and dashboard development.

---

## 🎯 Objectives

* Understand customer purchasing behaviour.
* Identify high-performing product categories.
* Analyze customer demographics and spending patterns.
* Discover trends that influence purchasing decisions.
* Develop an interactive dashboard for decision-making.

---

## 🛠️ Tools Used

* Microsoft Excel
* Power BI
* Power Query
* DAX
* GitHub

---

## 📂 Dataset Information

The dataset contains customer shopping transaction details, including:

* Customer Information
* Gender
* Age
* Product Category
* Purchase Amount
* Payment Method
* Shopping Frequency
* Customer Preferences

---

## 🔄 Project Workflow

### 1. Data Cleaning (Excel)

* Checked for missing values.
* Removed duplicate records.
* Standardized column formats.
* Verified data consistency.

### 2. Data Transformation (Power Query)

* Imported Excel data into Power BI.
* Cleaned and transformed data.
* Created appropriate data types.

### 3. Dashboard Development

Built interactive dashboards to visualize:

* Total Sales
* Customer Distribution
* Product Category Performance
* Purchase Behaviour
* Spending Patterns
* Demographic Analysis

---

## 📊 Dashboard Features

### Executive KPIs

* Total Customers
* Total Sales
* Average Purchase Value
* Number of Transactions

---

### Customer Analysis

* Customer distribution by gender.
* Age group analysis.
* Shopping frequency analysis.

---

### Product Analysis

* Top-performing product categories.
* Category-wise sales contribution.
* Customer preferences by category.

---

### Behaviour Analysis

* Preferred payment methods.
* Spending behaviour trends.
* Shopping pattern insights.

---

## 🔍 Key Insights

* Certain product categories contributed significantly to overall sales.
* Customer preferences varied across demographic groups.
* Specific payment methods were used more frequently.
* Shopping behaviour differed among customer segments.
* Interactive filtering enabled deeper exploration of customer trends.
